# Simple API Todo List Project

## Description
A simple Rest API for Learning golang at [myskill.id](https://myskill.id)

## Getting Started

### Installation
```sh
git clone https://github.com/adhiva/simple-api
cd simple-api
go get
```

### Running Application
```sh
go run main.go
```

### Dependencies
* [Gin Golang](https://github.com/gin-gonic/gin)
